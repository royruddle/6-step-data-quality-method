# Project Authors
The original version of this method was published as A Practical Guide to Characterising Data and Investigating Data Quality (2024) by Roy Ruddle, James Cheshire and Sarah Johansson Fernstad (https://doi.org/10.5518/1481). This github repository contains the most up to date version of the method, which is now called *The 6-Step Data Quality Method*.

## Primary Author
- Roy A. Ruddle - University of Leeds (UK)

## Co-Authors
- James Cheshire - University College London (UK)
- Sarah Johansson Fernstad - Newcastle University (UK)
